## lazy-regex

A small library providing regular expressions in simple English

## Installation

  npm install lazy-regex --save

## Usage

  ```javascript
  const LazyRegex = require("../index.js");
  const LazyRegexp = LazyRegex();

  var params = 'Hello World',
      log = LazyRegexp.log(params),

  console.log();
  ```
